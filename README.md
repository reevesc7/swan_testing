# SWAN testing

A set of tests exploring soft-wired artificial neuron (SWAN) brain implementations

## SWAN brains

Reeves, C., Walter, A., & Bohm, C. (2025, October). SWAN Brains: a novel substrate for evolving and studying bio-inspired neural circuits. In *Artificial Life Conference Proceedings* 37 (Vol. 2025, No. 1, p. 13). One Rogers Street, Cambridge, MA 02142-1209, USA journals-info@ mit. edu: MIT Press. [Link](https://direct.mit.edu/isal/proceedings/isal2025/37/13/134030)

Soft-wired artificial neuron (SWAN) brain is a modular, evolvable framework
for studying the evolution of neural networks *in silico*.
The main contribution of the framework to the study of digital evolution
is the ability to substitute components of models built within it,
enabling comparative analysis among models.

## Testing SWAN

To the end of implementing models within SWAN brains,
this repository exists as a testing bed for SWAN brain components.
It includes utilities for testing models over parameter ranges and reporting results.
These can be used to identify the evolutionary viability of models
and reasonable ranges for static and evolvable parameters.
